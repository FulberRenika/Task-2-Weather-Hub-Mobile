# 🌦️ Weather Hub Mobile

A simple Weather Hub mobile application developed using Flutter as part of the CodTech Internship - Task 2.

## 📌 Project Overview

Weather Hub Mobile is a Flutter application that displays a clean and simple weather interface. It is designed as a basic weather app prototype with a modern UI and can be extended in the future by integrating real-time weather APIs.

## ✨ Features

- Clean Flutter UI
- Welcome Screen
- Weather Information Display
- Mobile Responsive Design
- Material Design
- Ready for Future Weather API Integration

## 🛠️ Technologies Used

- Flutter
- Dart
- Android Studio
- Visual Studio Code

## 🚀 How to Run

1. Clone the repository

git clone https://github.com/FulberRenika/Task-2-Weather-Hub-Mobile.git

2. Open the project in Android Studio or VS Code.

3. Get the Flutter packages

flutter pub get

4. Run the application

flutter run
